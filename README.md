# 🩺 診断の手引き PDF Downloader

小児慢性特定疾病情報センターの公式サイトから、各疾患の **「診断の手引き」** を自動巡回し、PDFとして一括保存するためのPythonスクレイピングツールです。

## 📌 概要
このスクリプトは、Seleniumを使用して以下の動作を自動化します。

1. 疾患群一覧ページから各疾患ページをスキャン。
2. 各ページにある「診断の手引き」のリンクを抽出。
3. Chromeの印刷機能（Headlessモード）を利用し、レイアウトを保持したままPDFとして出力。
4. ファイル名に「疾患ID」と「疾患名」を自動付与して整理。

## 🛠 技術スタック
* **Python 3.x**
* **Selenium**: ブラウザ操作用
* **WebDriver Manager**: Chromeドライバの自動管理用
* **Chrome DevTools Protocol (CDP)**: 高品質なPDF出力（印刷設定の制御）

## ✨ 主な機能
* **インテリジェント・リネーム**: サイト上のタイトルから不要な文言を除去し、OSで禁止されている記号（`/:*?"<>|`）を自動置換。
* **二重ダウンロード防止**: 既にファイルが存在する場合はスキップするため、再実行時の効率が良い。
* **高精度なPDF化**: 背景色や画像を維持したまま、印刷用レイアウトを反映して出力。

## ⚠️ 注意事項
* **サーバー負荷への配慮**: スクリプト内に `time.sleep()` を含め、サーバーに過度な負荷をかけない設計にしています。
* **著作権について**: 取得した「診断の手引き」の著作権は、各学会や小児慢性特定疾病情報センターに帰属します。データの取り扱いは各サイトの規約に従ってください。
