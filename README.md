# 🩺 診断の手引き PDF Downloader & Processor

小児慢性特定疾病情報センターの公式サイトから「診断の手引き」を自動収集し、さらに内容を最適化（不要セクションの削除）するためのツールセットです。

## 📌 概要
このプロジェクトは以下の2段階の処理を自動化します。

1. **収集 (`pdfhozon.py`)**: 疾患群一覧から各疾患ページを巡回し、Chromeの印刷機能を用いて高品質なPDFを保存します。
2. **加工 (`shouman_trimmer.py`)**: 保存したPDFを解析し、本文以降の「関連資料」などの不要なページを自動判定してカットします。

## 🛠 技術スタック
* **Python 3.x**
* **Selenium / WebDriver Manager**: Webスクレイピング・PDF出力用
* **PyMuPDF (fitz)**: PDFのテキスト解析・編集（ページ削除）用

## ✨ 主な機能

### 1. 自動収集 (Collector)
* **インテリジェント・リネーム**: 疾患IDと疾患名を自動取得し、OS禁止文字を置換して保存。
* **二重保存防止**: 取得済みファイルを検知してスキップする機能を搭載。

### 2. コンテンツ最適化 (Optimizer)
* **キーワード検知**: PDF内部のテキストをスキャンし、「関連資料」セクションの開始位置を特定。
* **スマート・トリミング**: 
    * キーワードがページ上部にある場合：そのページから末尾までを削除。
    * キーワードがページ中段以降にある場合：次のページから末尾までを削除。
* **一括処理**: 数百件のPDFを数秒で最適化し、必要な診断基準のみを残した軽量な資料を作成します。

## ⚠️ 注意事項
* **パスの設定**: 加工スクリプトを実行する際は、ソースコード内の `INPUT_DIR` および `OUTPUT_DIR` を自身の環境に合わせて書き換えてください。
* **サーバー負荷への配慮**: 収集スクリプトには `time.sleep()` を含んでいます。
* **著作権について**: 取得した資料の著作権は、各学会や小児慢性特定疾病情報センターに帰属します。
